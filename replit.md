# George Smith Landing Page

## Project Overview
A personal landing page for George T. Smith, a Business Analytics and Information Systems student. Built with plain HTML5 and CSS3 — no frameworks or build tools.

## Tech Stack
- **Languages:** HTML5, CSS3
- **Fonts:** Inter (Google Fonts)
- **Build System:** None (static site)
- **Package Manager:** None

## Project Structure
```
.
├── index.html       # Main entry point
├── PRD.md           # Product Requirements Document
├── STANDARDS.md     # Technical and design standards
└── README.md        # Project intro
```

## Running the Project
The site is served via Python's built-in HTTP server:
```
python3 -m http.server 5000 --bind 0.0.0.0
```

## Deployment
Configured as a **static** deployment — files are served directly from the root directory.
