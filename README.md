# George T. Smith — Personal Landing Page

A professional personal landing page for George T. Smith, a University of Iowa senior graduating in May 2026 with a BBA in Business Analytics & Information Systems and a minor in Sports Management. The page exists to bridge the gap between a traditional resume and a full professional identity, communicating George's career direction toward sports law, his personality, and his academic and work experience in one accessible place. It is designed for professional networking audiences including employers, professors, legal professionals, and sports organizations. Built with plain HTML5 and CSS3, the site is fast, accessible, and requires no framework or build tooling.

---

## Features

- Sticky navigation bar with smooth scroll links to each section
- Circular hero headshot with name and professional tagline
- Bio section highlighting academic background, study abroad experience, and career direction
- Skill tags grouped by Technical and Interpersonal categories using pill-style badges
- Three experience cards in a responsive two-column grid layout
- Contact section with labeled links to LinkedIn, GitHub, and email
- Fully responsive layout from 320px to desktop widths
- WCAG 2.2 Level AA accessible — descriptive alt text, logical heading hierarchy, keyboard navigable

## Contributing

This is a personal project, but if you'd like to suggest improvements, feel free to fork the repository and submit a pull request.

1. Fork the repository on GitHub
2. Clone your fork: `git clone https://github.com/GeorgeSmitty/george-smith-landing-page`
3. Create a new branch for your changes: `git checkout -b your-feature-name`
4. Make your edits to `index.html` and/or `stylesheet.css`
5. Commit your changes with a clear message: `git commit -m "Describe your change"`
6. Push to your fork: `git push origin your-feature-name`
7. Open a Pull Request on the original repository and describe what you changed and why
